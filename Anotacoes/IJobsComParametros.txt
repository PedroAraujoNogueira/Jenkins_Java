1. Para definirmos quais classes queremos executar na criação do build podemos usar
parametros. Para isso vá em:

1.1. configurar -> general -> Este build é parametrizado -> Adicionar parametrizado
1.2. Escolher o tipo de parametro que queremos adicionar.
1.3. Ir no tópico Build e trocar o comando shell. Por exemplo:
javac $NomeEscolhidoParaParametros.java
