1. Um dos pré réquisitos para instalar o Jenkins é instalar o Java(JDK). Para instalar o JDK no 
linux podemos usar o comando:
-> sudo apt install default-jdk
Esse comando também instalará o JRE:
Caso queira saber mais sobre a instalação do Java, pode seguir esse tutorial.
-> https://www.digitalocean.com/community/tutorials/how-to-install-java-with-apt-on-ubuntu-20-04-pt

2. Podemos instalar o Jenkins no Linux, Windows, MAC ou em containers. O link para a documentação 
da instalação no Linux é:
-> https://www.jenkins.io/doc/book/installing/linux/

3. Para iniciar o serviço do Jenkins use o comando:
systemctl start jenkins
E para verificar o status, digite: 
systemctl status jenkins

4. Com o serviço do Jenkins já startado, podemos acessá-lo através da porta 8080.
http://localhost:8080/

5. Para garantir que o Jenkins possa ser configurado de forma segura por um administrador, 
uma senha é escrita no arquivo /var/lib/jenkins/secrets/initialAdminPassword, então use no 
terminal o comando cat /var/lib/jenkins/secrets/initialAdminPassword para pegar a senha e 
digite a senha no espaço referente a ela no http://localhost:8080/, após isso clique em 
continue.

6. Instale os plugins sugeridos.

7. Após os plugins serem instalados, você será direcionado para a página onde criará o 
primeiro usuario administrador, preencha os campos da forma que achar melhor e clique em 
"Save and Continue" e "Save and Finish" na página seguinte.
